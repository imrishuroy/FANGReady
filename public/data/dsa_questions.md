### Non-Leetcode Problems (with description)

**Remove Bad Pairs from a String**

Remove adjacent characters where the same letter appears in different cases (e.g., `xX` or `Aa`). Continue removing until no such pairs remain. This is typically solved using a stack-like approach to process characters and cancel bad pairs efficiently.

---

**Problem About Interns and Flats Using a Greedy Approach**

Assign interns to flats optimally based on constraints (like minimizing distance or maximizing satisfaction). The idea is to sort both lists and greedily match them to achieve the best allocation under constraints.

---

**Song Shuffler**

Design a shuffle algorithm where every permutation of songs is equally likely. This is a classic problem solved using the Fisher–Yates shuffle to ensure uniform randomness.

---

**Convert Graph to Binary Tree with Alternating Colors**

Given a graph (tree-like), find a root such that when structured as a binary tree, nodes at each level share the same color and adjacent levels alternate colors. This involves BFS/DFS with validation of level-wise coloring.

---

**Find Root Node in Colored Tree with Pattern Constraints**

Similar to the previous problem, but with stricter color patterns (e.g., repeating sequences like RBW). The task is to identify a valid root that satisfies level-wise constraints across the tree using traversal and validation logic.
